## 3. PaperWriter 怎么接收这些数据

所有函数的输出汇总到项目 JSON 中，PaperWriter 的 `write_paper_from_project(project_id)` 自动读取：

```python
# 项目文件 (.science/projects/sci_xxx.json) 的完整结构：
{
    "project_id": "sci_xxx",
    "domain": "...",
    "hypothesis": {...},              # MingLi 输出
    "knowledge_gaps": [...],          # TanXi 输出
    "mechanism_report": {...},        # YanZhen 输出
    "papergraph_records": [...],      # ZhiZhi 输出
    
    # ★ 需要你们填充的：
    "experiment_protocol": {          # GeWu 的 design_experiment + define_baselines + define_metrics
        "datasets": [...],
        "baselines": [...],
        "metrics": [...],
        "falsification_criteria": "..."
    },
    "experiment_results": {...},      # CodeEngineer 的 execute_code 输出
    "analysis_report": {...}          # MingBian 的 analyze_results 输出
}
```
