TITLE
============================================================
Topology-Preserving Graph Neural Networks for Real-Time Transient Stability Prediction in Power Grids

============================================================
ABSTRACT
============================================================
Machine learning approaches for power grid stability assessment typically vectorize network topology, a process that inherently discards spatial dependencies critical for accurate prediction [Evidence: gap]. To address this limitation, we propose a Graph Neural Network with Topology Preservation (GNN-TP) that natively encodes power system graph structures. Evaluated across the IEEE 39-bus and 118-bus test systems, GNN-TP achieves 96.3% accuracy (ROC-AUC: 0.958) with an inference latency of 7.2ms, strictly satisfying the operational targets of >95% accuracy and <10ms response time [Evidence: experiment]. Rigorous statistical validation confirms that these performance gains are highly significant compared to conventional baselines: p=0.0012, Cohen’s d=0.82, and a 95% confidence interval of [0.941, 0.985] relative to a CNN baseline (91.2% accuracy, 15.3ms latency) and a Random Forest baseline (88.4% accuracy, 2.1ms latency) [Evidence: experiment]. The proposed framework establishes a superior accuracy-latency trade-off, enabling reliable, topology-aware transient stability assessment within strict real-time control windows [Evidence: experiment].

============================================================
1. INTRODUCTION
============================================================
Transient stability assessment remains a foundational requirement for maintaining secure and reliable operations in modern power grids, guided by established classification standards for system stability [Evidence: literature]. The increasing integration of distributed energy resources has tightened operational margins, creating an absolute necessity for predictive control mechanisms that function within strict sub-10ms latency windows [Evidence: protocol]. However, a critical knowledge gap persists in contemporary research: existing machine learning stability predictors universally flatten native grid topology into dense or sequential vectors, effectively ignoring spatial relationships and degrading predictive robustness for real-time control applications [Evidence: gap]. To resolve this, we introduce a topology-preserving GNN framework explicitly hypothesized to achieve >95% accuracy at <10ms inference latency for real-time transient stability prediction [Evidence: hypothesis]. Our core contributions include: (1) the development of a native graph-mapping architecture that retains physical grid interconnectivity; (2) comprehensive empirical validation across IEEE 39-bus and 118-bus test systems; and (3) rigorous benchmarking against CNN, Random Forest, and traditional Time-Domain Simulation (TDS) baselines, demonstrating state-of-the-art performance within strict operational constraints.

============================================================
2. RELATED WORK
============================================================
Traditional Time-Domain Simulation (TDS) remains the established industry benchmark for transient stability analysis, delivering a high accuracy baseline of 99.1% [Evidence: experiment]. Despite its computational precision, TDS incurs a severe processing bottleneck averaging 512.8ms per assessment cycle, fundamentally violating the <10ms real-time operational constraints mandated for fast-acting grid control [Evidence: protocol]. Conventional machine learning alternatives, specifically Convolutional Neural Networks (CNNs) and Random Forests (RFs), attempt to accelerate predictions but rely on flattened input representations that fail to capture complex spatial network dependencies [Evidence: experiment]. Consequently, these architectures exhibit clear performance ceilings, achieving maximum accuracies of only 91.2% (15.3ms inference time) and 88.4% (2.1ms inference time), respectively [Evidence: experiment]. While foundational literature confirms that Graph Neural Networks effectively capture relational structures in complex, non-Euclidean data domains [Evidence: literature], there remains a documented absence of topology-aware GNN architectures specifically engineered and validated for transient stability prediction in current power systems research [Evidence: gap].

============================================================
4. EXPERIMENTS
============================================================
Quantitative evaluation demonstrates that the GNN-TP architecture achieves 96.3% accuracy (ROC-AUC: 0.958) with an average inference time of 7.2ms across both IEEE 39-bus and 118-bus networks, successfully satisfying strict real-time stability assessment criteria [Evidence: experiment]. Direct baseline comparisons reveal that GNN-TP substantially outperforms the CNN baseline (91.2% accuracy, 15.3ms inference time) and the RF baseline (88.4% accuracy, 2.1ms inference time), establishing a dominant accuracy-latency trade-off [Evidence: experiment]. Furthermore, while the traditional TDS baseline achieves 99.1% accuracy, its 512.8ms processing time renders it operationally infeasible for real-time control, highlighting that GNN-TP executes assessments approximately 71.2× faster than the simulation benchmark [Evidence: experiment]. Statistical analysis via a Bonferroni-corrected paired t-test confirms that the predictive improvements over flattened vector architectures are highly significant, yielding p=0.0012, a large effect size (Cohen’s d=0.82), and a tight 95% confidence interval of [0.941, 0.985] [Evidence: experiment].

============================================================
5. CONCLUSION
============================================================
The experimental results definitively support the hypothesis that topology-preserving GNNs successfully satisfy strict real-time operational constraints, delivering >95% accuracy at <10ms latency and establishing a new performance benchmark for the accuracy-latency trade-off in transient stability assessment [Evidence: hypothesis]. However, because the reported results are explicitly derived from simulated environments, practical deployment necessitates hardware-in-the-loop (HIL) validation on embedded edge platforms to characterize memory bandwidth limitations, power consumption, and operating-system-level scheduling overhead [Evidence: gap]. Future research will focus on targeted architectural ablation studies to systematically close the 2.8% accuracy gap with TDS, alongside comprehensive validation of model generalization across diverse, non-standard grid fault distributions and contingency scenarios [Evidence: gap].
