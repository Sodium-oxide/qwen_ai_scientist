TanXi Gap 探索要求：优先发现的问题包括遗传变异到功能表型再到药物暴露和毒性/获益之间的因果链断裂点；某些生物标志物是否真正具有预测治疗效果的能力；单次基线多组学预测与治疗过程中克隆演化、耐药、免疫状态变化之间是否存在断链；类器官、PDX 或功能检测的体外响应与临床响应不一致的原因；个体化细胞/核酸产品的制造过程中是否存在未验证的因果链等。
